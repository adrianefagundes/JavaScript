Iniciando os estudos em JavaScript

Nesse exercício conheci algumas funções da linguagem JS.

Primeiramente, deve-se utilizar a tag <script> para adicionar as funções da linguagem.

  • Para abrir um alerta antes do carregamento da página, utiliza-se o comando window.alert().
  • Se quiser que essa janela necessite de uma confirmação, utiliza-se o comando window.confirm().
  • Para que o usuário tenha que digitar alguma informação no promt, utiliza-se window.prompt().
