O que é DOM?

DOM é um acrônimo para Document Object Model e ele permite que o desenvolvedor navegue na árvore DOM (na hierarquia de elementos dentro do documento html) de maneira mais rápida e simplificada.

A fim de fazer um teste de modificação de texto com JavaScript, será adicionado este parágrafo e mais um, logo abaixo. Este último será colorido de azul após o comando:

var p1 = window.document.getElemntsByTagName('p')[2]
p1.style.color = 'blue'

Método de seleção de elementos

  • Por Marca (tag):

    • getElementsByTagName()

  • Por ID

    • getElementByID()

  • Por Nome

    • getElementsByName()

  • Por Classe

    • getElementsByClassName()

  • Por Seletor

    • querySelector()
    querySelectorAll()

A DIV TAMBÉM PODE SER MODIFICADA DA MESMA FORMA

var d = window.document.getElementById('mensagem')
d.style.background = 'beige'

QuerySelecor()

Quando se está selecionando por ID, pode-se fazer a seleção da forma acima OU utilizando os seletores, como no exemplo abaixo:

var d = window.document.querySelector(div#msg)

Nesse caso teriamos a mesma função exercida do que o que foi feito anteriormente com getElementById.

Com o querySelector utiliza-se a sintaxe do CSS.
